Welcome to the application page. Here you can find, install and edit applications from the market.


  • In the list on top of this view, you will find the applications, you already installed. Click on them, if you want to use them right away. Click on the gear icon at the right to change settings for this application.
  • In the second list, we recommend some applications for you which we think might be interesting for you. Click on them to see details.
  • Click on the market link on the navigation to find more applications and visualizations.
  • You can install new applications from the detail view by clicking on the install button. To uninstall them again, click on the gear icon next to the application to show the detail view again. Then click the uninstall button.
  • Also in the detail view you can set circles and spaces to applications you have installed already. This will assign all new record entries produced by this app to the corresponding circle or space. In the detail view for visualizations you can create new spaces with this visualization right away by a simple click on a buttton.
  • Finally in the myApps page you will find all the applications and visualizations you have already installed.
