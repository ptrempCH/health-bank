Welcome to the message inbox page. Here you can send and receive messages from other users.


  • In the inbox you will find the latest messages you received. Click on them to get the full content.
  • Click on the button on the top right to write a new message. There select the user you would like the message to send to, write a title for the message and then the message itself before sending it.
  • From the message detail view you can directly reply to a message by clicking on the corresponding page.
  • In the action box on top of the screen, you find a direct link to the inbox. If the message icon turns blue and shakes, click on it, because you received a new message.
